## üì¶ Getting Started

### Prerequisites
- Node.js 18+
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Rohan143-mp/Shree-Self-Adhesive.git
   cd Shree-Self-Adhesive
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## üö¢ Deployment

This project is configured for easy deployment to **GitHub Pages**.

To deploy the latest version:
```bash
npm run deploy
```
This command will:
1. Build the static project into the `out/` directory.
2. Push the contents of the `out/` directory to the `gh-pages` branch.

## üì± Offline Support

Thanks to its PWA configuration, the site remains accessible even without an internet connection once it has been visited for the first time. It automatically caches critical assets using a Service Worker.
